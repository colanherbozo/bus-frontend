# Bus Frontend - React

Aplicación frontend desarrollada con React que consume la API de buses.

## 🛠️ Tecnologías

* React 18
* JavaScript
* Bootstrap
* Fetch API

## Funcionalidades

* Listado de buses en tabla
* Paginación
* Consumo de API REST
* Visualización de detalle de bus

## Ejecución

1. Clonar repositorio
2. Instalar dependencias:

npm install

3. Ejecutar:

npm start

## Conexión API

La aplicación consume:

http://localhost:8080/bus
